<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Dear Chameleon users,</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">The summer is almost in full swing, and we're frantically getting stuff done so we can kick back and just rest on our laurels from May until August… just kidding. Expect lots more fun stuff to be announced this summer!</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Run your workshops on Chameleon with Jupyter. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Chameleon's JupyterHub Trovi artifact makes it simple to set up a multi-user Jupyter Notebook environment on a bare metal node, allowing you to provide a user-friendly interface to Chameleon's computing capabilities. This was deployed successfully for a few workshops (such as the </span><a href="https://www.anl.gov/article/students-connect-stem-skills-to-realworld-challenges-through-the-2020-virtual-big-data-camp" style="text-decoration: none;"><span style="">2020 ANL Big Data Camp</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> and the </span><a href="https://csinparallel.org/csinparallel/workshops/Virtual20/program.html" style="text-decoration: none;"><span style="">2020 CSInParallel Summer Virtual Workshop</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">) and we recently added more features to the Trovi artifact to make the workshop or classroom experience even better, namely, support for pre-configured user environments, easier onboarding of users, and the ability to provide datasets to all users. </span><a href="https://www.chameleoncloud.org/experiment/share/1" style="text-decoration: none;"><span style="">Read more about the features on Trovi</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> and give it a spin! Keep an eye out later this month for a Tips &amp; Tricks blog post with more details about how to run a workshop on Chameleon.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Support for part-time availability for CHI-in-a-Box associate sites. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">One common question that site operators ask when evaluating </span><a href="https://github.com/chameleoncloud/chi-in-a-box" style="text-decoration: none;"><span style="">CHI-in-a-Box</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, the packaging of Chameleon that you can deploy to manage and provide your own hardware via Chameleon, is "can I block out periods of time where the hardware is allocated elsewhere?" The answer is yes, and now it's significantly easier to do this with the Doni enrollment service, which comes pre-packaged with CHI-in-a-Box. Using the Doni CLI client, you can not only enroll your hardware inventory easily, but you can also set availability windows, to indicate temporary maintenance or perhaps mark the entire site as unavailable (not reservable) for a period of time. Check out the </span><a href="https://github.com/ChameleonCloud/chi-in-a-box/wiki/Hardware-management#updating-a-nodes-availability" style="text-decoration: none;"><span style="">CHI-in-a-Box wiki</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> for more information on Doni and the capabilities it offers.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">New Jupyter notebook stitching tutorial (create wide-area dedicated layer 2 links!). </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">The </span><a href="https://www.chameleoncloud.org/experiment/share/33" style="text-decoration: none;"><span style="">Chameleon User Guide: Networking</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> Trovi artifact continues to grow. We have added a tutorial that demonstrates layer 2 networking stitching between Chameleon sites using the </span><a href="https://python-chi.readthedocs.io/en/latest/" style="text-decoration: none;"><span style="">Python API</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. Come learn how to create isolated high bandwidth network connections for your Chameleon experiments!</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Improved support for sign in via federated identity. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">As you know, your host institution can change. Names can change. Life comes at you fast! We've worked on making some changes to the new </span><a href="https://chameleoncloud.readthedocs.io/en/latest/user/federation.html" style="text-decoration: none;"><span style="">federated single sign-on</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> to accommodate this reality. Before, it was pretty easy to confuse the system into thinking it needed to make a new account for you, leading to you having multiple Chameleon accounts, which may </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: italic; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">sound</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> delightful, but was a bit annoying in practice. We sent the login system to do a post-bacc year in pattern recognition and it's doing a better job of tracking down your account now.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Re-home our retired phase 1 hardware!</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> Announcing the </span><a href="https://bit.ly/3vRRJSj" style="text-decoration: none;"><span style="">Chameleon Legacy Hardware Program</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">! We're offering our retired hardware to members of the community interested in joining the Chameleon family as operators of a </span><a href="http://bit.ly/3tx9GVr" style="text-decoration: none;"><span style="">Chameleon Associate Site</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. Learn all about the Chameleon Legacy Hardware Program and </span><a href="http://bit.ly/3nI9PDc" style="text-decoration: none;"><span style="">apply for servers</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> by submitting a short application. Applications are encouraged by 05/16.</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> Application available by filling out this short </span><a href="https://bit.ly/3nI9PDc" style="text-decoration: none;"><span style="">Google Form</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. Applications encouraged by 05/16, when we plan to start initial discussions with future partners.</span></p>

<p dir="ltr" style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Seeking comment on IoT hardware investment. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Chameleon is expanding to support Edge to Cloud experiments later this year. For this reason we will be purchasing a variety of IoT devices that our users will be able to provision in a manner similar to our nodes. We expect this functionality to be available in late summer/fall on a preview basis for early users only. A rough outline of our current purchase plans is </span><a href="https://bit.ly/3ef2iZp" style="text-decoration: none;"><span style="">here</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> and we seek your feedback. If you would like to provide feedback on what devices to purchase and/or are interested in being an early user, please fill out this </span><a href="https://bit.ly/3uh2fC4" style="text-decoration: none;"><span style="">form</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. </span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Upcoming changes to floating IP usage. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">So, turns out you all REALLY like </span><a href="https://www.chameleoncloud.org/blog/2019/02/27/save-planet-use-fewer-ips/" style="text-decoration: none;"><span style="">Floating IPs</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. So much so that we're having trouble keeping them in stock. To make this situation a bit better, we're going to start reclaiming IPs that are not reserved in advance as part of a lease. Fun fact: you can reserve IPs, and should in fact do so! In May we'll start automatically reclaiming Floating IPs not tied to a lease with more regularity. Please refer to </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/reservations.html#creating-a-lease-to-reserve-floating-ips" style="text-decoration: none;"><span style="">our documentation</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> for instructions on how to reserve a public IP address via the CLI (or the </span><a href="https://python-chi.readthedocs.io/en/latest/modules/lease.html#chi.lease.add_fip_reservation" style="text-decoration: none;"><span style="">Python API</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> if you prefer!)</span></p>

<p> </p>

<p style="text-align: center;"><img alt="Chameleon hardware" src="/media/uploads/2015/09/28/chameleon_hardware.jpeg"> </p>

<p>We are happy to announce that the new Chameleon OpenStack KVM cloud is available! This cloud spans three additional racks of <a href="https://www.chameleoncloud.org/about/hardware-description/">Chameleon hardware</a> with configuration similar to our bare metal racks.The cloud is configured with the most recent OpenStack release (Kilo) and offers the same set of OpenStack services as the KVM cloud available up to now.</p>

<p><br>
All images, snapshots, volumes, and volumes snapshots have been copied today from the old hardware to the new cloud. However, no instances have been migrated.</p>

<p> </p>

<p>If you are already a Chameleon user, follow the <a href="https://www.chameleoncloud.org/docs/user-guides/openstack-kvm-user-guide/">OpenStack KVM User Guide</a> to access the new cloud; if you are new to Chameleon get started in <a href="https://www.chameleoncloud.org/docs/getting-started/">three simple steps</a>.<br>
<br>
If required, you can continue using the old hardware, but please note that it will become unavailable after October 9.<br>
<br>
We also won't be able to migrate user data (images, snapshots, volumes or volume snapshots) created on the old cloud after today. If you do need to migrate data at a later date, <a href="https://www.chameleoncloud.org/docs/user-guides/openstack-kvm-user-guide/#downloading_and_uploading_data">you can use the OpenStack clients</a>.<br>
<br>
As usual, don't hesitate to contact us via the help desk if you have any questions.</p>

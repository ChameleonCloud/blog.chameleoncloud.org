<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Dear Chameleon users,</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">We held back the changelog a day just so that you wouldn't think we were foolin' you with all of these updates we squeezed out in March! The theme here is "making things simpler" for you. Complexity is never really removed, just shifted, and that's one of the main goals of Chameleon at the end of the day: shifting complexity away from you so that you can spend more brain cycles on your own codes and ideas.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Get started with CHI-in-a-Box via the QuickStart Jupyter notebook. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">We continued to improve the documentation and user experience of </span><a href="https://github.com/chameleoncloud/chi-in-a-box" style="text-decoration: none;"><span style="">CHI-in-a-Box</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, the packaging of Chameleon you can deploy at your own host institution. We now provide a </span><a href="https://www.chameleoncloud.org/experiment/share/35" style="text-decoration: none;"><span style="">QuickStart </span></a><span style="text-decoration: none;"><span style="">notebook</span></span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, which guides you through the process of installing CHI-in-a-Box on a leased bare metal node on Chameleon. It's Chameleon on Chameleon! The appliance is in the form of an Jupyter Notebook tutorial, which walks you through the process of reserving the required resources and then configuring CHI-in-a-Box. At the end you'll have your own little Chameleon cloud! (A cloud of one node, which I suppose is technically the smallest cloud possible.)</span></p>

<p><span id="docs-internal-guid-7a1d2c2a-7fff-a2b7-6833-a468117a651e" style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Announcing Bring Your Own Device, in preview. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">We're happy to announce an early preview of a powerful new service that is included in CHI-in-a-Box. It allows you to enroll your own hardware into a Chameleon site and then manage your inventory and its availability (e.g., to support part-time availability of hardware) in one place. Check out the </span><a href="https://github.com/ChameleonCloud/chi-in-a-box/wiki/Hardware-management" style="text-decoration: none;"><span style="">CHI-in-a-Box wiki</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> for more information about how to enroll and update hardware via this preview service!</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Simpler CLI usage from within your instances.</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> Using the </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/cli.html" style="text-decoration: none;"><span style="">Chameleon CLI</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> from within a node was always a bit tricky to set up, but got trickier when Chameleon supported federated authentication. We've now made the CLI automatically authenticate using the project the instance was launched under. This should make the CLI "just work"--no more wrangling </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/cli.html#the-openstack-rc-script" style="text-decoration: none;"><span style="">RC files</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> or fiddling with authentication parameters!</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Add users to projects via email, finally.</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> It's about time, right? Before, PIs had to add users to a Chameleon project by inputting the target user's username. It's now possible to additionally add users by inputting the email address tied to the user's account (this is visible from the user profile page.) Hope that makes things simpler for you PIs!</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Jupyter upgraded to JupyterLab 3.0 release. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">We took some time to upgrade </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/jupyter.html" style="text-decoration: none;"><span style="">Chameleon's Jupyter interface</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> this month and deployed the latest version of JupyterLab for all users. JupyterLab 3.0 adds several usability improvements, notably a built-in table of contents view and a streamlined Simple Interface, which make it easier to read and refer to larger Notebook documents, in particular if they are being </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/sharing.html" style="text-decoration: none;"><span style="">shared via Trovi</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> to other users. Please refer to the </span><a href="https://blog.jupyter.org/jupyterlab-3-0-is-out-4f58385e25bb" style="text-decoration: none;"><span style="">official release announcement</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> if you're curious to find out more.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Finally, a quick note that we're </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">deprecating the </span><a href="https://www.chameleoncloud.org/appliances/19/" style="text-decoration: none;"><span style="">Ubuntu 16 appliance</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. Ubuntu 16 is leaving LTS at the end of this month and we already provide Ubuntu 18 and 20 appliances; please switch to those if you haven't already!</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-0d4627b8-7fff-a49d-2cb9-c356abc4d3ef" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Great news in Chameleon-land!</span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-0d4627b8-7fff-a49d-2cb9-c356abc4d3ef" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">We went a trick-or-treating yesterday and were successful -- as a result we have a lot of goodies to share with you. </span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-0d4627b8-7fff-a49d-2cb9-c356abc4d3ef" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">New Chameleon Associate Site at Northwestern. </span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">A couple of months ago we announced </span><a href="https://www.chameleoncloud.org/blog/2018/09/05/august-2018-changelog/" style="text-decoration: none;"><span style="">the release of an alpha version of CHI-in-a-Box</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> -- a packaging of the CHameleon Infrastructure (CHI), i.e., our core testbed services, that allows interested parties to join Chameleon as a site or create their own Chameleon-like testbeds. We are now happy to announce that our partners at Northwestern immediately took advantage of this opportunity and created a Chameleon Associate Site at Northwestern. For now, the site offers only one node -- but it is equipped with Mellanox ConnectX-5 100GE network interface, not available anywhere else on Chameleon! These network cards allow you to experiment with large flows and will perform best using Chameleon’s </span><a href="https://www.chameleoncloud.org/appliances/50/docs/" style="text-decoration: none;"><span style="">Large Data Flow Appliance</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">, re-tuned for 100Gbps as described </span><a href="https://www.chameleoncloud.org/blog/2018/10/19/transferring-large-data-flows-chameleon/" style="text-decoration: none;"><span style="">in our blog a couple weeks ago</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. Since CHI-in-a-Box is still in an experimental phase, the new site is available to Early Users only: if you are interested please </span><a href="https://www.chameleoncloud.org/user/help/" style="text-decoration: none;"><span style="">open a ticket via the helpdesk</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> to request access and  include “CHI@NU” in the subject of the ticket to help us expedite your request. Once your account on the new site has been activated, you will be able to reach the site at </span><a href="https://sl-ciab.northwestern.edu/" style="text-decoration: none;"><span style="">https://sl-ciab.northwestern.edu</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">.  </span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-0d4627b8-7fff-a49d-2cb9-c356abc4d3ef" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Experiment Précis Service (alpha)</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. Did you ever find yourself using the Linux history command to figure out what exactly you did to solve a problem? We feel that testbeds -- where all of us are trying hard to make our experiments reproducible -- should help this effort by providing a similar functionality. The </span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: rgb(255, 255, 255); font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Chameleon </span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Experiment Précis (CEP) service (précis means “summary”) helps you track your experiments by recording events of your experiment setup performed on the testbed (e.g., creating leases, hardware used, creating instances and setting up networks, or monitoring information collected by  Gnocchi). You can then request a summary of this information and -- for now -- you will be given simply a record of events that occured during your experiment.</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: rgb(255, 255, 255); font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> Since this is new experimental functionality, at this time the experiment precis is only available at CHI@UC site. For more information about CEP client installation and usage, please consult our </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/ep.html" style="text-decoration: none;"><span style="">documentation</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: rgb(255, 255, 255); font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> -- and please do let us know what you think about this new feature and how you’d like to improve it.</span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-0d4627b8-7fff-a49d-2cb9-c356abc4d3ef" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Better Chameleon Jupyter Notebooks. </span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Last month, </span><a href="https://www.chameleoncloud.org/blog/2018/10/01/chameleon-changelog-september-2018/" style="text-decoration: none;"><span style="">we announced a new way to use Chameleon</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> -- within your very own hosted Jupyter notebook. This month we’re happy to announce a new feature to make it easier to collaborate on experiments with others. You can now save your notebook files in a </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/swift.html" style="text-decoration: none;"><span style="">Swift</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> container shared within your project. This means that anybody in your project can open your Notebook, see your process, and build upon it. We also put together a </span><a href="https://vimeo.com/297210055" style="text-decoration: none;"><span style="">screencast</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> to illustrate how the Jupyter Notebook can help you set up more reproducible experiments - take a few minutes and check it out! And of course, to learn more, visit our </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/jupyter.html" style="text-decoration: none;"><span style="">documentation</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">.</span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-0d4627b8-7fff-a49d-2cb9-c356abc4d3ef" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Upgrade of FPGA Compiler.</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> In our </span><a href="https://www.chameleoncloud.org/blog/2018/09/05/august-2018-changelog/" style="text-decoration: none;"><span style="">August changelog</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> we announced a hardware contribution of the new Terasic DE5a-Net FPGA made by one of our users to the University of Chicago site. We now support building applications for this board (in addition to the already supported Nallatech boards) on our FPGA build node. As always, consult </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/fpga.html" style="text-decoration: none;"><span style="">our documentation</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> on how to use it. </span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-0d4627b8-7fff-a49d-2cb9-c356abc4d3ef" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Usability improvements.</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> We made it easier to see the physical host name of the host running an instance; users can see it in the Instances section of the dashboard. The name links out to a page listing the hardware details for that node. </span></b></p>

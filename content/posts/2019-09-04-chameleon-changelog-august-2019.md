<p dir="ltr" id="docs-internal-guid-2a70bdba-7fff-f172-e9bf-944a15a820d3" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Great news in Chameleon-land!</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Hopefully those working in the U.S. enjoyed your Labor Day festivities! We’ve been hard at work ourselves over the past month and return from the break with some new goodies for you.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">For those of you interested and learning more about the advanced networking capabilities of Chameleon, there is an excellent webinar on September 10, which you can </span><a href="https://bit.ly/chameleonsept10" style="text-decoration: none;"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(17, 85, 204); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: underline; vertical-align: baseline; white-space: pre-wrap;">register for here</span></a><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">New Jupyter appliance</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. </span><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">By now you are hopefully familiar with </span><a href="https://jupyter.chameleoncloud.org/" style="text-decoration: none;"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(17, 85, 204); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: underline; vertical-align: baseline; white-space: pre-wrap;">Chameleon’s Jupyter environment</span></a><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">, which allows you to conveniently experiment within a dedicated Jupyter server. However, this shared platform is currently insufficient for larger-scale data analysis or experimentation requiring large volumes of data. So you will be pleased to learn that we have released a new </span><a href="https://www.chameleoncloud.org/appliances/72/" style="text-decoration: none;"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(17, 85, 204); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: underline; vertical-align: baseline; white-space: pre-wrap;">Jupyter appliance in our appliance catalogue</span></a><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">, which will enable you to launch notebooks, just as you would now, but within a bare metal server environment that you control, enabling you to utilize the full storage, compute, and network capabilities of that node, rather than the portioned resources allotted in the shared environment. You should explore this option particularly if you require local access to data sets larger than 1GB. Check out the </span><a href="https://www.chameleoncloud.org/appliances/72/docs/" style="text-decoration: none;"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(17, 85, 204); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: underline; vertical-align: baseline; white-space: pre-wrap;">documentation</span></a><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> in the appliance catalogue link to learn more.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Import Jupyter experiments from GitHub</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. </span><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Additionally, we have made it easier to import existing GitHub repositories inside the notebook environment. This means that if you have access to a GitHub repository you can with one click automatically provision a notebook server just for that repository with all required dependencies already installed. To try out this feature, head on over to our </span><a href="https://github.com/chameleoncloud/CC-CentOS7" style="text-decoration: none;"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(17, 85, 204); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: underline; vertical-align: baseline; white-space: pre-wrap;">CC-CentOS7</span></a><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> repository and click the “launch on Chameleon” buttons, visible in the readme--this will open the repository’s code in a new Jupyter server so you can explore how we at Chameleon build our bootable base disk images. This is just one additional benefit of exploring Chameleon’s Jupyter integration.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Learn more about our associate site at Northwestern. </span><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">You can now discover the various hardware available for use at the CHI@NU associate site at Northwestern in </span><a href="https://www.chameleoncloud.org/hardware/" style="text-decoration: none;"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(17, 85, 204); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: underline; vertical-align: baseline; white-space: pre-wrap;">our hardware catalog</span></a><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. While you can browse the available resources, access still requires invitation–click the </span><a href="https://www.chameleoncloud.org/chi-nu" style="text-decoration: none;"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(17, 85, 204); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: underline; vertical-align: baseline; white-space: pre-wrap;">CHI@NU button</span></a><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> to learn how to request access!</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Lastly, here are some important deadline reminders for conferences this month (remember to <a href="https://www.chameleoncloud.org/about/frequently-asked-questions/#toc-how-should-i-acknowledge-chameleon-in-my-publications-"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(17, 85, 204); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: underline; vertical-align: baseline; white-space: pre-wrap;">acknowledge</span></a> and </span><a href="https://www.chameleoncloud.org/about/frequently-asked-questions/#toc-how-should-i-reference-chameleon-"><span style="font-size: 11.5pt; font-family: Arial; color: rgb(17, 85, 204); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: underline; vertical-align: baseline; white-space: pre-wrap;">cite Chameleon in your paper</span></a><span style="font-size: 11.5pt; font-family: Arial; color: rgb(29, 28, 29); font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> if you have used the testbed in your research!)</span></p>

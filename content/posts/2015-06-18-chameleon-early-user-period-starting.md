title: The Chameleon Early User period is starting!
---

<p>We are happy to announce the beginning of the Chameleon Early User period.</p>

<p>The Early User period has been created to allow for exercising the new hardware under realistic usage scenarios prior to public availability. Participants in the program will be able to gain unlimited access to new Chameleon hardware while Chameleon operators will be able to assess how the new hardware behaves under real-life conditions.  </p>

<p><a href="https://www.chameleoncloud.org/user/early-user-program/program/1/">Sign up for the Early User Program</a> to get access. Once approved you will be able to access the Early User Guide and follow its instructions to use the new Chameleon hardware.</p>

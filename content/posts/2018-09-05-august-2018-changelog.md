<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-c315a962-7fff-2931-4cd4-1a79bfac5f5f" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Great news in Chameleon-land!</span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">We worked hard all through the last month and right after the Labor Day we are happy to announce the fruit of those labors:  </span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><br>
<b id="docs-internal-guid-c315a962-7fff-2931-4cd4-1a79bfac5f5f" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Managing the number of nodes assigned to a lease</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. While, it has always been possible to </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/reservations.html#extending-a-lease" style="text-decoration: none;"><span style="">manage the time allocated to Chameleon leases</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> -- in other words, prolong or shorten them -- up till now it has not been possible to manage them in space, i.e, change the number of reserved nodes. Users who found themselves with insufficient number of nodes for an experiment had to release their current allocation and create a bigger lease. We now provide capabilities to update the number of nodes in a lease. For advance reservations that haven't yet started, the node count can be increased or decreased. For reservations already started, new nodes can be added (the ability to drop nodes from an active lease will be added in the future). Read our documentation for detailed information on </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/reservations.html#changing-the-number-of-nodes-of-a-lease" style="text-decoration: none;"><span style="">how to update the number of reserved nodes</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. </span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-c315a962-7fff-2931-4cd4-1a79bfac5f5f" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Hardware Contribution.</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> We are excited to announce a hardware contribution made by one of our users: the new </span><a href="https://www.intel.com/content/www/us/en/programmable/solutions/partners/partner-profile/terasic-inc-/board/arria-10-device-family---de5a-net--fpga-development-kit.html" style="text-decoration: none;"><span style="">Terasic DE5a-Net FPGA</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> board has been installed and is available for use by all in experiments on Chameleon. The board is </span><a href="https://www.terasic.com.tw/cgi-bin/page/archive.pl?Language=English&amp;CategoryNo=231&amp;No=970#https://www.terasic.com.tw/attachment/archive/970/image/DE5a-Net_45d.jpg" style="text-decoration: none;"><span style="">powered by a Altera Arria 10 GX and has ultra low-latency, straight connections to four external 40G QSFP+ modules</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> and provides new capabilities for studying hardware architectures. The hardware node is </span><a href="https://www.chameleoncloud.org/hardware/node/sites/uc/clusters/chameleon/nodes/581c40ec-c957-4399-b62a-d03d114dfb7c/" style="text-decoration: none;"><span style="">available for reservation at the UC site</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> and can be launched with the existing </span><a href="https://www.chameleoncloud.org/appliances/32/" style="text-decoration: none;"><span style="">FPGA appliance</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. For now, users should compile their OpenCL codes on their workstation with a Quartus installation using their own license before uploading to the FPGA node -- in the future, we will make the compiler available on the Chameleon compile nodes. </span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-c315a962-7fff-2931-4cd4-1a79bfac5f5f" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">CHI-in-a-box</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. We have released an alpha version of </span><a href="https://chameleoncloud.readthedocs.io/en/latest/getting-started/faq.html#what-is-chi-in-a-box" style="text-decoration: none;"><span style="">CHI-in-a-box</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> -- a packaging of the CHameleon Infrastructure, i.e., core testbed services. </span><a href="https://chameleoncloud.readthedocs.io/en/latest/getting-started/faq.html#what-is-in-chi-in-a-box-alpha" style="text-decoration: none;"><span style="">The alpha version</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> package contains only a part of the infrastructure (in particular, the discovery services are not yet packaged) and is meant for adventurous users operating in constrained conditions. If you would like to explore the alpha CHI-in-a-box, please contact us at </span><a href="mailto:contact@chameleoncloud.org" style="text-decoration: none;"><span style="">contact@chameleoncloud.org</span></a></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-c315a962-7fff-2931-4cd4-1a79bfac5f5f" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">Publishing appliances</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. We heard from multiple users that while it was very easy to make an appliance public, it was hard to add any useful information to the published appliance (such as a line of description, information about documentation, mailing lists, or support) -- limiting the usefulness of such sharing in practice. We therefore now provide a mechanism </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/images.html#publishing-images-to-the-appliance-catalog" style="text-decoration: none;"><span style="">that makes sharing this additional information with other users much easier</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">: simply click the Images tab on the left of Chameleon GUI, and select Publish to Appliance Catalog from the dropdown menu to the right of listed images. You will be presented with a web form that allows you to enter comments about your appliance -- the resulting entry will be published in the appliance catalog that can be accessed directly off our </span><a href="http://www.chameleoncloud.org/" style="text-decoration: none;"><span style="">main page</span></a><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">.</span></b></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><b id="docs-internal-guid-c315a962-7fff-2931-4cd4-1a79bfac5f5f" style=""><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">User interface extensions</span><span style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">. We added a feature whereby the PIs of Chameleon projects can see how many SUs each of their projects is consuming.</span></b></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Great news in Chameleon-land!</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">You’re probably a bit leery about any news <a href="https://en.wikipedia.org/wiki/April_Fools%27_Day">today</a>, and rightly so, but I can tell you that these new features are no joke.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Floating IPs are now reservable (in preview!) </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">As the </span><a href="https://www.chameleoncloud.org/blog/2019/02/27/save-planet-use-fewer-ips/" style="text-decoration: none;"><span style="">number of public floating IPs available in Chameleon is limited</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, we need to make sure they are used efficiently, just like our hardware resources. It thus made perfect sense to extend our reservation capabilities to include floating IPs, given our recent good experience with </span><a href="https://www.chameleoncloud.org/blog/2019/02/01/chameleon-changelog-january-2019/" style="text-decoration: none;"><span style="">making the coveted ExoGENI VLANs reservable in advance</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">You can now reserve one or several floating IPs for your experiment using the lease system. Reserved floating IPs automatically appear in your project when their reservation starts, and disappear when it ends. Rest assured, your workflow will remain unchanged for now. Only a limited range of floating IPs was made reservable: you can still request some directly from the system using </span><a href="https://chameleoncloud.readthedocs.io/en/latest/getting-started/index.html#associating-an-ip-address" style="text-decoration: none;"><span style="">the old way</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. Over time, we will make more floating IPs usable only via reservation, but will likely let projects still use a small number of floating IPs without reserving them, space permitting.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">This new feature is available as a preview and is currently only supported by the </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/reservations.html#provisioning-and-managing-resources-using-the-cli" style="text-decoration: none;"><span style="">blazar CLI</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. We are working on supporting it in our web interface. See </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/reservations.html#creating-a-lease-to-reserve-floating-ips" style="text-decoration: none;"><span style="">our documentation</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> for more details on how to reserve your first floating IP!</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">An easier way to save/load experimental data is here! </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">When logged into an instance provisioned with a Chameleon-supported image (like our </span><a href="https://www.chameleoncloud.org/appliances/1/" style="text-decoration: none;"><span style="">CentOS7</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> or </span><a href="https://www.chameleoncloud.org/appliances/19/" style="text-decoration: none;"><span style="">Ubuntu16.04</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> images), you will now see a directory called “</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: italic; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">my_mounting_point</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">” which is a already mounted to the Chameleon Object Store. This means that any data you’ve already uploaded will just be a `cp` away. Also, you can save your experiment results here (tarballs work great for this) so they survive even if your instance is terminated when your lease ends. We also updated the mounting tool, which now is called “</span><a href="https://github.com/ChameleonCloud/cc-cloudfuse" style="text-decoration: none;"><span style="">cc-cloudfuse</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">”. The new tool allows you to configure your Chameleon credentials using your existing </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/cli.html#the-openstack-rc-script" style="text-decoration: none;"><span style="">RC files</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. Click </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/swift.html#mounting-object-store-as-a-file-system" style="text-decoration: none;"><span style="">here</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> to learn more about mounting Chameleon object store as a file system.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Explore new tutorials in our Jupyter environment. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Hopefully by now you’ve tried out </span><a href="https://jupyter.chameleoncloud.org" style="text-decoration: none;"><span style="">our hosted Jupyter Notebook environment</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. If not, now’s a great time to start! When you log in to your Notebook server you should notice a new folder magically dropped in there called “</span><a href="https://github.com/chameleoncloud/notebooks" style="text-decoration: none;"><span style="">notebooks</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">”. This is a special Git repository we maintain of useful examples and tutorials for advanced features on Chameleon. Have a look and play around with some of the Bash-based notebooks that should help familiarize you with the Chameleon CLI and give you some powerful tools.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Project administration improvements. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Last month </span><a href="https://www.chameleoncloud.org/blog/2019/03/01/chameleon-changelog-february-2019/" style="text-decoration: none;"><span style="">we announced a long-requested feature</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> where the system will now allow you to rename your project to something besides a soulless numeric ID prefixed by “CH-” via a nickname. We spruced this up a bit, allowing you to see your project nickname in the </span><a href="https://www.chameleoncloud.org/user/dashboard/" style="text-decoration: none;"><span style="">Dashboard</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, and also worked to ensure we properly notify you if a nickname is already taken.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Some big changes incoming.</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> We are doing a lot of spring cleaning and are preparing to release an upgrade of the foundational layer to Chameleon, </span><a href="https://www.openstack.org/" style="text-decoration: none;"><span style="">OpenStack</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. This change should bring numerous performance benefits as well as additional features. We will be announcing new features as they are polished, tested, and integrated via these changelog entries, which we know you read and </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: italic; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">love</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">.</span></p>

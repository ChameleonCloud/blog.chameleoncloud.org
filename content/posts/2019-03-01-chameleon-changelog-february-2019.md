<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Great news in Chameleon-land!</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">We were busy meeting with you all at the </span><a href="https://www.chameleoncloud.org/user-meeting-2019/" style="text-decoration: none;"><span style="">Chameleon User Meeting</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> in Austin, but in between gorging ourselves on Texan BBQ and Mexican food, we conjured up some new things for you to enjoy. (If you typically stop reading at the first paragraph, then just skip to filling out our </span><a href="https://chameleoncloud.typeform.com/to/OZPz3i" style="text-decoration: none;"><span style="">new user survey for 2019</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, which will help us prioritize what shiny new hardware to purchase and what features to work on.)</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Easier debugging of BYOC networks.</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> If you’ve tried to create a network using a custom OpenFlow controller </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: italic; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">a la</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> our </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/networks/networks_sdn.html" style="text-decoration: none;"><span style="">Bring-Your-Own-Controller (BYOC)</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> capability, you may have encountered some issues debugging your OpenFlow controller (they can be finicky!) To assist you in this, we’ve published some live metrics for our Corsa DP2000 switches for all Chameleon users to use and refer to. You can log in to the </span><a href="https://grafana.chameleon.tacc.utexas.edu/" style="text-decoration: none;"><span style="">Chameleon Grafana</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> server to see raw packet counts across switch ports--useful for knowing if traffic is properly flowing out of your controller or across your network! We hope to add more useful dashboards to this Grafana interface in the future so you have better insight into the goings-on of the various Chameleon sites. To learn more, check out </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/networks/networks_sdn.html#debugging-your-controller" style="text-decoration: none;"><span style="">the documentation on this feature</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">More features for your Jupyter notebooks. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">You can now write a Notebook using the Bash scripting language: very handy if you don’t want to use the Python APIs to interact with Chameleon and would rather just use the CLI that you’re used to. In case you don’t believe us, try out one of our </span><a href="https://github.com/ChameleonCloud/notebooks/blob/master/tutorials/BYOC/BYOC.ipynb" style="text-decoration: none;"><span style="">example Notebooks describing how to set up a BYOC networking experiment</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">! Additionally, you can now easily check out Git repositories into your notebook environment, allowing you to save your Notebooks to GitHub to take advantage of version control, or perhaps to pull in other files from other remote repositories. Stay tuned for a blog post highlighting some potential workflows integrating Jupyter Notebooks with GitHub!</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">CH-CH-CHanges (to your project names).</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> Ever wished you could change your project name from its default machine-readable “CH-XXXXXX” designation to something meaningful to you, like the name of your cat (Mr. Meowmix) or perhaps your favorite food additive (Red 40)? Well, consider your wish granted. You can now add a project nickname in the </span><a href="https://www.chameleoncloud.org/user/projects/" style="text-decoration: none;"><span style="">Chameleon Dashboard</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. This nickname will then show up throughout the user interface and you can also use it when authenticating via the CLI. You may need to update your </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/cli.html#the-openstack-rc-script" style="text-decoration: none;"><span style="">RC files</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> to include the updated project nickname.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Experiment automation at TACC</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. </span><a href="https://www.chameleoncloud.org/blog/2019/02/01/chameleon-changelog-january-2019/" style="text-decoration: none;"><span style="">Last month we announced</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> the ability to set up your lease such that it launches an appliance for you automatically when the lease starts -- but it was only available at the UC site. This feature is now also available at TACC. To learn more about how you can use this feature to automate your experiment workflows, check out </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/complex.html#advanced-reservation-orchestration" style="text-decoration: none;"><span style="">this section of our documentation</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Turning up the Heat on Ubuntu. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">We had a long-standing limitation where you could not launch </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/complex.html#introduction" style="text-decoration: none;"><span style="">complex appliances</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> on Chameleon if you were using an Ubuntu base image. This was not obvious and certainly not ideal, so we sorted this out this month. Now if you’re using either the </span><a href="https://www.chameleoncloud.org/appliances/1/" style="text-decoration: none;"><span style="">CC-CentOS7</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> or </span><a href="https://www.chameleoncloud.org/appliances/19/" style="text-decoration: none;"><span style="">CC-Ubuntu16</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> image, you can use any complex appliance </span><a href="https://www.chameleoncloud.org/appliances/" style="text-decoration: none;"><span style="">in our catalog</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. Limitless freedom!</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Usability improvements for PIs. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Sometimes you want to use Chameleon for a class or for a workshop. It can be really annoying to manage the members of the project in that case, because our UI would only show some garbled auto-generated usernames instead of anything useful, like their name or email address. Those days are over! We now show you more relevant information in this part of the UI to help you out. Also, we made a change to alleviate any potential </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: italic; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Zuteilungfristablaufsangst </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">(an archaic German word for “fear of one’s allocation expiring”): we now show you if your pending allocation renewal has already been approved, so you know that your project will remain active.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">A new survey for a new year. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">After gathering feedback during the User Meeting, we wanted to follow up with a few specific questions and give those of you who were not able to attend the meeting a chance to provide input about what hardware we should buy, and what we should prioritize. You can access the survey </span><a href="https://chameleoncloud.typeform.com/to/OZPz3i" style="text-decoration: none;"><span style="">here</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. </span><span style="font-size: 11pt; font-family: Arial; color: #222222; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Please provide your feedback by </span><span style="font-size: 11pt; font-family: Arial; color: #222222; background-color: #ffffff; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">March 15</span><span style="font-size: 11pt; font-family: Arial; color: #222222; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. Thank you in advance for your participation!</span></p>

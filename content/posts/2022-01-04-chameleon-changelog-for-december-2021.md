<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Dear Chameleon users,</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">The massive gears of time lock in another lumbering rotation and we find ourselves just beyond the opening of the year 2022. Hope this message finds all of you well, and best wishes for the new year from all of us at Chameleon! We have some exciting things to kick off this year, so let's just get to it.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Early preview of Trovi Public API!</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> In late 2020 </span><a href="https://www.chameleoncloud.org/blog/2020/12/31/2020-end-year-changelog/" style="text-decoration: none;"><span style="">we announced Trovi</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, the reproducibility platform that enables you to experiment with Chameleon via Jupyter Notebooks, and then package and share your work with others easily. Over the last year Trovi has been used to reproduce various </span><a href="https://www.chameleoncloud.org/blog/2020/11/23/reproducing-solid-state-drive-simulator-research-results-chameleon/" style="text-decoration: none;"><span style="">systems</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> </span><a href="https://www.chameleoncloud.org/blog/2020/11/01/chameleon-and-reproducibility-linnos-case-study/" style="text-decoration: none;"><span style="">experiments</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, </span><a href="https://www.chameleoncloud.org/blog/2021/11/22/using-chameleon-for-artifact-evaluation/" style="text-decoration: none;"><span style="">evaluate research artifacts for SOSP'21 and SC'21</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, and we recently implemented the </span><a href="https://www.chameleoncloud.org/blog/2021/10/05/chameleon-changelog-for-september-2021/" style="text-decoration: none;"><span style="">Daypass feature</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, which allows anyone to temporarily reproduce an experiment even if they do not have a Chameleon allocation (helpful for conference evaluation in particular!)</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">We are happy to announce that we are open-sourcing Trovi and working to transform it into a more general-purpose tool for reproducibility! As the first step, we've defined an API specification, which you can read on the </span><a href="https://chameleoncloud.gitbook.io/trovi/" style="text-decoration: none;"><span style="">early preview documentation site</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> for the Trovi Public API. Later this month we will be standing up a dedicated API service that you can use to query and update the artifact repository. Eventually we will similarly open-source our extensions to Jupyter that make saving and retrieving artifacts to/from Trovi a breeze, so keep tuned for that as well :)</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">(More) new hardware at CHI@UC! </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">The hardware continues to roll in. We're happy to announce that the Xilinx cards </span><a href="https://www.chameleoncloud.org/blog/2021/11/01/chameleon-changelog-for-october-2021/" style="text-decoration: none;"><span style="">we teased in November</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> are now installed and ready for experimentation at CHI@UC. Two </span><a href="https://www.xilinx.com/products/boards-and-kits/alveo/u280.html" style="text-decoration: none;"><span style="">Alveo U280</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">s are attached to nodes </span><a href="https://chameleoncloud.org/hardware/node/sites/uc/clusters/chameleon/nodes/89e48f7e-d04f-4455-b093-2a4318fb7026" style="text-decoration: none;"><span style="">P3-CPU-038</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> and</span><a href="https://chameleoncloud.org/hardware/node/sites/uc/clusters/chameleon/nodes/926a9c99-3b27-45a7-818c-e6525b9ce89c" style="text-decoration: none;"><span style=""> P3-CPU-042</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">; you can also filter for these nodes in our hardware browser by using the advanced filters and finding the "Board Model" option, as shown below.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">We've also installed a new consumer SSD into node </span><a href="https://chameleoncloud.org/hardware/node/sites/uc/clusters/chameleon/nodes/7ed407a7-98dd-4708-bf32-9e0ab50c9f68/" style="text-decoration: none;"><span style="">P3-SSD-010</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> for additional storage experimentation. There will be more consumer and enterprise SSDs arriving soon, and we'll announce in which nodes they are installed. As a reminder, some of our specialized nodes, such as those containing FPGAs and additional SSDs, carry a higher charge towards your allocation than some of our less "special" hardware.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">CHI-in-a-Box point release. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">In the past months we've been making several refinements to CHI-in-a-Box, the packaging of the Chameleon infrastructure that you can deploy yourself. In particular, we've updated the </span><a href="https://github.com/ChameleonCloud/chi-in-a-box/wiki/QuickStart" style="text-decoration: none;"><span style="">Quick Start guide</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> with more guidance about how to configure the networking at the host site, which has been a common source of questions. We also integrated support for out of band power monitoring via IPMI, something that was implemented to power the </span><a href="https://www.chameleoncloud.org/blog/2021/11/12/summary-of-indyscc-2021-chameleon-usage/" style="text-decoration: none;"><span style="">IndySCC</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> competition! On an administrative node, we are trialing </span><a href="https://github.com/ChameleonCloud/chi-in-a-box/discussions" style="text-decoration: none;"><span style="">GitHub Discussions</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> to answer questions and announce any minor updates to CHI-in-a-Box in the future, so do check that out and ask questions there; operators are standing by. For more details, please refer to the </span><a href="https://github.com/ChameleonCloud/chi-in-a-box/releases/tag/v2022-01.03" style="text-decoration: none;"><span style="">official release page on GitHub</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">!</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">End of life for CentOS8 appliance. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">CentOS 8 reaches EOL on December 31, 2021. As CentOS Stream is its successor, we've released a new </span><a href="https://chameleoncloud.org/appliances/103/" style="text-decoration: none;"><span style="">CC-CentOS8-stream</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> and its </span><a href="https://chameleoncloud.org/appliances/104/" style="text-decoration: none;"><span style="">CUDA version</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. They have UEFI boot support and can be used on the new UC nodes. We will keep the CC-CentOS8 and CC-CentOS8-CUDA11 images in the </span><a href="https://www.chameleoncloud.org/appliances/" style="text-decoration: none;"><span style="">appliance catalog</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> but they will no longer be updated and will not receive improvements to Chameleon-provided tooling.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Other things that happened</span></p>

	<li style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">When reserving </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/networks/networks_stitching.html#configuring-a-stitchable-network" style="text-decoration: none;"><span style="">stitchable networks</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, you must now provide a "stitch_provider" constraint on the network reservation request instead of specifying "provider_network"--please refer to the </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/networks/networks_stitching.html#configuring-a-stitchable-network" style="text-decoration: none;"><span style="">documentation</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> for an example CLI invocation.</span></li>
	<li style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">It is possible to make reservations for any resource without specifying any constraints; in the past this would mean you would get some random hardware item from our inventory, which wasn't ideal because sometimes you could get an ARM architecture node, or something otherwise exotic from your perspective. We've added some different defaults now, so if you make a lease without constraints you'll get one of our "commodity" x86 nodes.</span></li>
	<li style="font-size: 11pt; font-family: Arial; color: rgb(0, 0, 0); background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Squashed an annoying bug in the </span><a href="https://jupyter.chameleoncloud.org/" style="text-decoration: none;"><span style="">Chameleon Jupyter environment</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> where you could get caught in a logout refresh loop!</span></li>

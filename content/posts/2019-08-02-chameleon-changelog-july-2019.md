<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Great news in Chameleon-land!</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">The summer is heating up--at least in Chicago--and here we are working on making the system better so you don’t have to! (We suggest using any newfound time floating in a pool in a large duck-resembling inner tube whilst sipping a cool drink.)</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">CHI-in-a-Box beta release out now! </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Last December </span><a href="https://www.chameleoncloud.org/blog/2019/01/03/chameleon-changelog-december-2018/" style="text-decoration: none;"><span style="">we announced</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> the second alpha release of CHI-in-a-Box. Since then, we have dramatically improved  how Chameleon works under the hood, first by moving to a container-based infrastructure, then by upgrading the entire system to the latest OpenStack releases. In the process we looked at CHI-in-a-Box and decided to change things up: instead of releasing and maintaining a separate packaged version of Chameleon independently, we decided to open-source our own deployment. This expands the scope of CHI-in-a-Box to include additional monitoring, maintenance tools, operational runbooks, and more. Our lessons learned can now be properly passed on to you! We hope that this improves the long-term health of CHI-in-a-Box as a program and makes it easier for you to consider Chameleon at your own institution. Have a look at the </span><a href="https://github.com/ChameleonCloud/chi-in-a-box" style="text-decoration: none;"><span style="">official GitHub repository</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, and contact us with any questions.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">New MPICH appliance available for all experimenters</span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. Good news for parallel computing researchers! The MPICH development team at Argonne National Laboratory contributed </span><a href="https://www.chameleoncloud.org/appliances/71/" style="text-decoration: none;"><span style="">a new appliance</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> with the latest version of MPICH (3.3.1) installed. MPICH is a high-performance and widely portable implementation of the Message Passing Interface (MPI) standard. To learn more about MPICH, please visit </span><a href="https://www.mpich.org/" style="text-decoration: none;"><span style="">the official website</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. This appliance is built on top of the latest version of </span><a href="https://www.chameleoncloud.org/appliances/1/" style="text-decoration: none;"><span style="">CC-CentOS7</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, so it has all Chameleon utilities installed, such as cc-snapshot, cc-cloudfuse, and the metrics collection agent.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Ubuntu18.04? Yes. CUDA10? Also yes. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Hot on the heels of </span><a href="https://www.chameleoncloud.org/blog/2019/07/02/chameleon-changelog-june-2019/" style="text-decoration: none;"><span style="">our announcement last month</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> of a new Ubuntu18.04 appliance, this month we released a </span><a href="https://www.chameleoncloud.org/appliances/70/" style="text-decoration: none;"><span style="">new CUDA 10.1 appliance for Ubuntu18.04</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">This appliance can be launched on the GPU-enabled hardware at CHI@TACC.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Support for updating complex reservations. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">When we expanded the Chameleon reservation system earlier this year to include </span><a href="https://www.chameleoncloud.org/blog/2019/04/01/chameleon-changelog-march-2019/" style="text-decoration: none;"><span style="">Floating IPs</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> and </span><a href="https://www.chameleoncloud.org/blog/2019/02/01/chameleon-changelog-january-2019/" style="text-decoration: none;"><span style="">VLANs</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">, we made it possible for you to reserve all the resources required for an experiment in one lease (and we </span><a href="https://www.chameleoncloud.org/blog/2019/05/02/chameleon-changelog-april-2019/" style="text-decoration: none;"><span style="">updated our GUI</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> to prove it!) One thing was still ahead of us though: allowing updates to complex reservations, such as changing the amount of Floating IPs needed, or requesting a different VLAN, or adjusting the amount of nodes in one of these complex leases. That limitation is now gone, enabling you to have more flexibility when making your leases. We encourage you to try out the new reservation types, in particular Floating IPs, which are relevant to virtually all types of experiments. Check out the </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/reservations.html" style="text-decoration: none;"><span style="">documentation</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> to learn more.</span></p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Faster response to allocation modifications. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">In the past, there would be some delays between adding a user to a project and them being able to actually access Chameleon, because the change took a bit to roll out to all of our systems. We took a look at this long-standing issue this month and have made some changes to speed this up significantly, which should reduce problems you (or, perhaps students in a class you’re teaching) might have with not being able to log in after project members are updated. If you find you ever do have questions about your allocation, first check our </span><a href="https://www.chameleoncloud.org/about/frequently-asked-questions/#toc-project-and-allocation-management" style="text-decoration: none;"><span style="">FAQ</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> to see if it answers your question, but always feel free to send us a ticket on our </span><a href="https://www.chameleoncloud.org/user/help/" style="text-decoration: none;"><span style="">Help Desk</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: #ffffff; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> - that’s what it’s there for.</span></p>

<p>Dear Chameleon users,</p>

<p>This month we are excited to announce a brand new Chameleon site, better Trovi metrics, and updated stitching documentation.</p>

<p><strong>New site, CHI@IIT!</strong> The federation of Chameleon sites is growing larger this month, with a new site at <a href="https://www.iit.edu/">Illinois Tech (IIT)</a> in Chicago. Like <a href="http://chi.evl.uic.edu">CHI@EVL</a> which came online in February, <a href="https://chameleon.cs.iit.edu">CHI@IIT</a> received Haswell nodes from CHI@UC as a part of <a href="https://www.chameleoncloud.org/blog/2021/05/03/chameleon-legacy-hardware/">Chameleon’s Legacy Hardware Program</a>. This site has <a href="https://chameleoncloud.org/hardware/">80 nodes</a>, making it a great place to conduct experiments where you want to use many nodes of the same type.</p>

<p><strong>New Trovi Metrics.</strong> You might be wondering what the little icons and the numbers next to <a href="https://chameleoncloud.org/experiment/share/">artifacts</a> on our <a href="https://chameleoncloud.readthedocs.io/en/latest/technical/sharing.html">sharing portal</a> Trovi. Previously, we showed the number of total launches and number of versions for an artifact. Now we’ve added more, including the number of unique users who launched the artifact and the number of users who ran some part of the artifact in Jupyter. To learn more about each specific metric, we added a tooltip that appears when you hover over an icon. We hope that these updates make it easier to find helpful artifacts, and are useful to artifact authors. </p>

<p><strong>Updated Fabric Stitching Documentation.</strong> A few months ago, we announced the ability to stitch experiments between <a href="https://fabric-testbed.net/">Fabric</a> and Chameleon at CHI@UC, if you have a project on both testbeds. This month, we’ve updated our <a href="https://chameleoncloud.readthedocs.io/en/latest/technical/networks/networks_stitching.html">documentation</a> to explain more about this process, in addition to the <a href="https://chameleoncloud.org/experiment/share/9284120f-3436-41f3-9e82-238e0628ec6c">Trovi artifact</a> we originally announced. We still have future plans to integrate further with Fabric, and you can expect more examples in the future.</p>

<p>As always, if there are any questions you have about testbed features, please <a href="https://chameleoncloud.org/user/help/ticket/new/">let us know</a>!</p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Dear Chameleon users,</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Wow, we're already well into October, as it turns out! We lost track of time a bit, as we've been focused on doing a massive upgrade of the CHI@UC infrastructure to support some cool new hardware that should be released to you shortly. Good things come to those who wait, after all. Thanks for your patience; we're happy to say that we've got some pretty exciting things to announce in the meanwhile!</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Announcing Daypass: Chameleon for a day! (preview) </span><a href="https://chameleoncloud.org/experiment/share/" style="text-decoration: none;"><span style="">Trovi</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> has been the place to share your experiments with Chameleon, but for another user to launch these notebooks, they had to be on an active project. This meant that reviewers or other researchers interested in your research couldn’t easily reproduce your experiment and analysis. That's why we are rolling out Daypass, which is a way to provide limited access to Chameleon resources specifically for the purpose of reproducing research artifacts. Owners of a Trovi artifact can now link them with an existing project and enable “reproducibility requests" via the Share menu. If enabled, below the “Launch” button on the artifact page there will be a button to “Request Daypass", like this:</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"><span style="border: none; display: inline-block; overflow: hidden; width: 624px; height: 324px;"><img height="324" src="https://lh4.googleusercontent.com/hxQxw2rQ3o3lY0zAhoa0hR7I3S0s8dwvage8Ii1VyBu1RwNocc6KJSqiMe3-VVqo7z3pmIAsl5f1TTyAuXUSjCQZXQU6o33LbLM28JXzWekJExDALq7Mj6h1oM6QGHn_bxdlGius=s0" style="margin-left: 0px; margin-top: 0px;" width="624"></span></span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">This button is basically magic: it allows a user to submit a request to the managers of the artifact’s project, which, if accepted, grants the user access to all of Chameleon for a few hours (or however long it takes to reproduce the artifact as specified by its author). </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/sharing.html#using-daypass" style="text-decoration: none;"><span style="">See our documentation</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> for a detailed explanation, and how to get started.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">You can now designate a PI delegate on Chameleon projects! </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">One of the roles of a Chameleon project PI is to manage membership of their project(s). However, this can be an arduous task, especially for PIs who are teachers and running classes where many students need to be added. PIs have asked (for quite a long time!) about being able to delegate this responsibility of project membership management to a trusted project member. We're very happy to say that this is now possible! PIs can now assign the Manager or Member (default) role to the project members. The Managers of the project have the ability to add and remove other members. Please read our user documentation on </span><a href="https://chameleoncloud.readthedocs.io/en/latest/user/project.html#manage-user-roles" style="text-decoration: none;"><span style="">how to manage roles</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> and </span><a href="https://chameleoncloud.readthedocs.io/en/latest/user/project.html#manage-users" style="text-decoration: none;"><span style="">how to manage membership</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> to learn more!</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Nuke it from orbit; it's the only way to be sure. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">Oops! You just uploaded some of your research to Trovi, but it wasn't ready yet. This is not a big deal, because anything you save in Trovi is private by default, and thus nobody will see it. But, sometimes you just want a big "undo" button--we've now built in a way to entirely delete one of your Trovi artifacts, including all versions. Please, use responsibly, and as always, </span><a href="https://chameleoncloud.readthedocs.io/en/latest/technical/sharing.html#editing-artifacts" style="text-decoration: none;"><span style="">refer to the docs for more info</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">.</span></p>

<p dir="ltr" style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"> </p>

<p><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 700; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">New release of Jupyter Notebook environment with Python 3.9. </span><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;">We updated the default Jupyter Notebook environment available on </span><a href="https://jupyter.chameleoncloud.org" style="text-decoration: none;"><span style="">Chameleon's JupyterHub</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> to support Python 3.9. Along with this upgrade, a few new features were added thanks to upstream Jupyter developers, notably support for a </span><a href="https://jupyterlab.readthedocs.io/en/stable/user/debugger.html" style="text-decoration: none;"><span style="">built-in debugger</span></a><span style="font-size: 11pt; font-family: Arial; color: #000000; background-color: transparent; font-weight: 400; font-style: normal; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre; white-space: pre-wrap;"> for Python and other supported kernels!</span></p>
